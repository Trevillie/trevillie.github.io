About Me

Research Interest

I'm broadly interested in topics within the range of computer security and privacy. My past work has mainly focused on security and privacy issues of emerging technologies, namely mobile applications and machine learning systems. For the former, I worked on Android application obfuscation and mobile browser privacy. For the latter, I studied adversarial examples and testing techniques for deep learning systems.

Currently, I'm working on machine learning security under the supervision of professor Hao Chen.
